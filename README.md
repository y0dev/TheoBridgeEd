# TheoBridge Education

Problem: Disparities in educational opportunities and resources for students from different socio-economic backgrounds.
Solution: Establish scholarship programs, provide access to technology for remote learning, and improve teacher training in underprivileged areas.

## :open_file_folder: Folder structure

- `components` every component involved with the backend
- `config` project configuration
- `dev_data` dummy data for database
- `middleware` all of the middleware components that are used in the project
- `public` frontend src code
- `utils` utilities components used throughout the API
- `views` pug files for the frontend website
